# 🤖 Privacy-First AI Chatbot (Local LLM with Ollama)

## 📌 Overview
This project is a fully local AI chatbot built using LangChain and Ollama (LLaMA3).  
It runs entirely on your machine, ensuring data privacy and zero external API dependency.

## 🚀 Features
- 🧠 Local LLM (LLaMA3 via Ollama)
- 🔒 Privacy-focused (no data sent to cloud)
- ⚡ Fast responses with local inference
- 🖥️ Simple Streamlit UI
- 📊 LangChain tracing with LangSmith

## 🛠️ Tech Stack
- Python
- LangChain
- Ollama (LLaMA3)
- Streamlit
- LangSmith
